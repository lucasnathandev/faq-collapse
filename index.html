Frequently Asked Questions

Why should i use const instead let in JS?

Because const hold data safer than let because let is variable and const is constant data.

Is HTML an programming language?

No, it's not a programming language. It's a markup language for browsers.

Where do i write JS code into HTML files?

You need to create into the HTML file an script tag.

How do i style my webpage to look fancy?

You can use CSS for it. There are two nice ways to do it: creating style tag into HTML or creating an CSS file.
